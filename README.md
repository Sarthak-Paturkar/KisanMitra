# KisanMitra - AI Farm Intelligence Platform

An AI-powered crop price prediction and farm intelligence platform built with Streamlit.

## Setup Instructions

1. Ensure you have Python 3.9+ installed.
2. Install the requirements:
   ```bash
   pip install -r requirements.txt
   ```
3. Add your Anthropic API Key to the `.env` file for the KisanBot to work.
4. Run the application:
   ```bash
   streamlit run app.py
   ```
